# Pulsar

Pulsar is an opinionated, self-host oriented, open source crypto portfolio manager.

## Main Features
- High security: self host the porfolio, own your data; You never need to connect your wallets.
- Multi chain support
- No limit on the number of wallets/accounts you can add
- Use your own keys: pay only for your specific usage
- Highly configurable: customize your wallet experience (snapshots etc...)
- Easily customizable/extendible
